# iSlide

iSlide 是一款基于 PowerPoint 的专业演示设计插件（Windows），内置海量模板、图标、图示、配色等资源库，帮助用户快速制作美观的 PPT。

> **注意**：iSlide 以 PowerPoint 插件形式运行，需要已安装 Microsoft Office（PowerPoint）。

## 安装

从官网下载安装包并运行：

1. 前往 [iSlide 官网](https://www.islide.cc/) 下载最新安装程序（`.exe`）
2. 运行安装程序，按提示完成安装
3. 重启 PowerPoint，菜单栏将出现 **iSlide** 选项卡

> 安装完成后需注册账号并登录，免费账号可使用基础功能，付费计划解锁完整资源库。

## 主要功能

| 功能模块 | 说明 |
|----------|------|
| **主题库** | 内置数千套专业 PPT 模板，支持一键应用并自动统一风格 |
| **图示库** | 提供逻辑图、流程图、矩阵图等多种可编辑矢量图示 |
| **图标库** | 覆盖多种风格的矢量图标，支持关键词搜索并直接插入 |
| **图片库** | 正版商用图片资源，可直接在 PPT 中搜索并插入 |
| **配色方案** | 预设配色库，一键替换幻灯片的主色调 |
| **智能排版** | 自动对齐、等距分布、批量统一字体大小等排版工具 |
| **PPT 瘦身** | 压缩图片、清理隐藏元素，减小文件体积 |
| **裁剪图片** | 将图片裁剪为自定义形状，支持批量操作 |

## 使用方法

安装后在 PowerPoint 菜单栏点击 **iSlide** 选项卡，各功能模块以面板形式展开：

- **资源库**（主题/图示/图标/图片）：搜索并双击或拖入幻灯片
- **设计排版**：选中元素后点击对应工具（对齐、等距、统一大小等）
- **PPT 瘦身**：`iSlide → 工具 → PPT 瘦身`，按提示压缩

> 大部分操作支持撤销（`Ctrl + Z`），可放心尝试。
