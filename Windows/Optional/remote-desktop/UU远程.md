# <img src="../../../.media/png/uu-remote.png" width="60" height="60" alt="uu logo"> 网易UU远程

网易UU远程是网易推出的一款远程桌面工具，面向中国用户优化了网络链路，在国内网络环境下连接速度与稳定性表现突出。支持 Windows 作为主控端和被控端，macOS 和 Android 作为被控端，适用于远程办公、协助家人解决电脑问题、远程游戏串流等场景。

> 个人用户免费使用，无广告、无付费墙；商业用途需参考官方授权说明。

## 安装

### Windows

```bash
# winget
winget install NetEase.UUYuan
```

也可从[官网](https://uuyc.163.com/)下载 `.exe` 安装包，安装后需注册或登录网易账号。

### macOS / Android

| 平台 | 获取方式 |
|------|---------|
| macOS | 从官网下载 `.dmg` 安装包 |
| Android | 各大应用商店搜索"UU远程"，或从官网下载 APK |

> 目前 iOS 端暂未上线，被控设备需为 Windows / macOS / Android。

## 快速入门

1. **注册账号**：在 Windows 主控端上安装 UU 远程并登录网易账号（支持手机号、邮箱等方式）。
2. **被控端安装**：在需要被远程控制的设备上安装对应客户端，登录同一账号。
3. **发起连接**：在主控端的设备列表中选中目标设备，点击"远程连接"即可。
4. **安全验证**：首次连接需输入被控端设置的安全密码，后续可设为自动登录。

## 核心功能

| 功能 | 说明 |
|------|------|
| 远程桌面 | 实时查看和操作远程设备桌面 |
| 文件传输 | 支持双向传输文件，拖拽即可完成 |
| 多显示器切换 | 被控端有多屏时可切换查看不同屏幕 |
| 自适应画质 | 根据网络状况自动调整画质与帧率 |
| 远程开机 | 配合硬件设备可实现远程唤醒（WOL） |
| 游戏串流 | 针对游戏场景优化延迟，支持手柄映射 |

## 使用提示

- **账号一致性**：主控端和被控端需登录同一网易账号，跨账号连接需通过设备码邀请。
- **网络环境**：UU 远程针对国内电信/联通/移动网络进行了链路优化，跨境连接时延迟可能上升。
- **安全密码**：建议在被控端设置独立的安全连接密码，不要与其他账号密码相同。
- **与游戏加速器的区别**：网易UU远程与"UU 网游加速器"是两款独立产品，功能不重叠，账号体系独立。
- **无人值守**：被控端设置为开机自启并登录后，可实现随时远程访问。
