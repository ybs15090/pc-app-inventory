# <img src="../../../.media/png/fences-logo.png" width="60" height="60" alt="Fences logo"> Stardock Fences

Windows 桌面图标整理工具，通过在桌面上创建可命名的分区（"栅栏"）对图标进行分组管理，告别杂乱桌面。

## 安装

从官网购买并下载安装包：

> 下载地址：[stardock-fences.com](https://stardock-fences.com/index.html)

安装后首次启动会引导设置初始布局，建议先对现有桌面图标做一次快照备份。

## 功能特点

- **图标分组（栅栏）**：在桌面创建阴影区域作为"栅栏"，将图标、文件、文件夹拖入分类，如工作、娱乐、学习
- **快速隐藏与显示**：双击桌面空白处隐藏全部图标，再次双击恢复；可设置特定栅栏不参与隐藏
- **自动组织规则**：按文件类型、名称、时间等定义规则，新图标自动归入对应栅栏；栅栏内图标可按多种条件排序
- **自定义外观**：调整栅栏背景色、透明度、圆角、边框及标签字体，打造个性化桌面风格
- **多页面与多屏支持**：创建多个桌面页面，鼠标拖至屏幕边缘切换；多显示器间可自由移动栅栏与图标
- **Folder Portals**：将任意文件夹镜像为桌面栅栏，无需打开资源管理器即可直接访问文件夹内容
- **快照与备份**：随时对桌面布局拍快照，防止误操作导致布局丢失
- **Peek 快速预览**：`Win + Space` 将栅栏浮于所有窗口之上，不最小化当前窗口也能快速访问
