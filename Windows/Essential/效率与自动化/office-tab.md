# Office Tab

Office Tab 是一款 Microsoft Office 插件，为 Word、Excel、PowerPoint 等应用添加类似浏览器的多标签页界面，让多个文档在同一窗口内以标签页形式并排管理，无需在任务栏反复切换窗口。

> 免费版支持基础标签功能；付费版（企业版）解锁更多自定义选项，个人日常使用免费版已足够。

## 安装

1. 访问 [官方下载页面](https://www.extendoffice.com/download/office-tab.html)
2. 下载安装包并运行，安装过程中选择需要启用标签的 Office 应用（Word / Excel / PowerPoint 等）
3. 安装完成后重启对应 Office 应用，顶部即出现标签栏

> 需要 Microsoft Office 2010 及以上版本，支持 Office 365 / Microsoft 365。

## 主要功能

| 功能 | 说明 |
|------|------|
| 多标签页 | 在同一窗口内以标签页形式管理多个文档 |
| 标签右键菜单 | 快速保存、另存为、关闭、重命名、复制路径等操作 |
| 拖拽排序 | 拖动标签调整文档顺序 |
| 双击关闭 | 双击标签快速关闭文档 |
| 标签颜色 | 为不同文档标签设置颜色以便区分 |
| 新建文档 | 点击标签栏末尾的 `+` 新建空白文档 |

## 使用方法

| 操作 | 方式 |
|------|------|
| 切换文档 | 单击对应标签页 |
| 关闭当前文档 | 双击标签 或 点击标签上的 `×` |
| 保存所有文档 | 右键任意标签 → 「保存所有」 |
| 关闭其他文档 | 右键标签 → 「关闭其他」 |
| 在文件夹中显示 | 右键标签 → 「打开文件位置」 |

**设置标签栏位置**：右键标签栏空白处 → 「Office Tab 中心」，可调整标签栏显示在窗口顶部或底部、标签样式、字体大小等。
