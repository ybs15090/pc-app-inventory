# <img src="../../../.media/png/quicker.png" width="60" height="60" alt="quicker logo"> Quicker

Quicker 是一款 Windows 平台的效率工具，将日常重复操作封装为"动作"，支持一键触发。用户可以根据自己的需求创建动作，也可以从共享库中安装他人分享的动作，覆盖文本处理、窗口管理、代码片段、剪贴板增强等多种场景。

## 安装

从[官网](https://getquicker.net/)下载安装包，注册账号后即可使用。

```bash
# winget
winget install Quicker.Quicker
```

## 核心概念

| 概念 | 说明 |
|------|------|
| 动作 | 一组可重复执行的操作序列 |
| 触发方式 | 鼠标中键、快捷键、轮盘菜单、文本指令等 |
| 共享库 | 社区分享的动作库，可一键安装使用 |
| 辅助服务 | 后台运行的剪贴板、搜索、窗口管理等功能 |
