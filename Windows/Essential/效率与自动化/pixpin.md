# <img src="../../../.media/png/pixpin.png" width="60" height="60" alt="pixpin logo"> PixPin

PixPin（Pixel Pin）是一款 Windows 平台的截图与贴图工具，以截图和贴图两大功能为核心，支持 OCR 文字识别、长截图、GIF 录制等扩展功能，目标是提升用户的日常工作效率。

## 安装

从[官网](https://pixpin.cn/)下载安装包，按向导完成安装即可。

## 使用

| 功能 | 说明 |
|------|------|
| 截图 | 默认快捷键截图，支持矩形/窗口/全屏等多种模式 |
| 贴图 | 截图后自动贴在屏幕上，方便对照参考 |
| OCR | 截图后可识别并复制图片中的文字 |
| 长截图 | 滚动窗口自动拼接长图 |
| GIF 录制 | 录制屏幕区域导出为 GIF |
