# <img src="../../../.media/png/pixpin.png" width="60" height="60" alt="Boxes logo"> PixPin

PixPin 的名字来源于"Pixel Pin"，简单来说是一款截图、贴图工具。PixPin 以截图和贴图两大功能为核心，扩展了丰富的实用功能，目标是提升用户的工作效率——无论你使用电脑从事何种工作，都可以借助 PixPin 更高效地完成。

- [官方网站](https://pixpin.cn/)
