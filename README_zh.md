# PC 软件合集
[English README](README_en.md)
## 概览

本仓库收录了适用于 PC 的推荐软件与工具清单，包含 Windows、Linux 和通用工具三个部分。

## Windows

适用于 Windows 的必装软件和生产力工具。

## Linux

目前主要是 Ubuntu 相关的软件和实用 Linux 工具。

## General

跨平台通用软件与配置。

## 使用方法

1. 克隆本仓库到本地（或者在线使用）。
2. 安装所需的软件。
3. 配置好环境。
4. 开始使用。

## 贡献

如果你对这个项目感兴趣，欢迎贡献自己的软件清单、配置建议或文档内容。

### 仓库文件格式规范
- 每个软件或工具都应该有一个独立的 Markdown 文件，命名为 `软件名称.md` 或 `软件用途.md`。
- 每个 Markdown 文件应该包含以下内容(推荐但不强制)：
  - 软件简介
  - 安装方法
  - 使用方法（简单说明即可）
  - 官方文档链接（如果有）
  - 其他相关资源链接（如教程、社区等）
- 如果要提供软件配置文件或其他相关资源，可以将其放在一个以软件名称（或用途）命名的文件夹中，并在 Markdown 文件中提供链接。
- 每个软件的 Markdown 文件应该放在对应的目录下（Windows、Linux 、Mac 或 General）。
