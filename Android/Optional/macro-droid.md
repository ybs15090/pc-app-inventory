# <img src="../../.media/png/MacroDroid.png" width="60" height="60" alt="macrodroid logo"> MacroDroid

MacroDroid 是一款 Android 平台的自动化工具，允许用户创建自定义宏（Macro），在特定事件触发时自动执行指定操作。无需编程基础，通过可视化界面即可配置"如果 → 那么"逻辑。

> 免费版支持的宏数量有限且包含广告。大多数核心功能无需 root，但部分高级操作需要 root 权限。

## 安装

通过 Google Play 搜索 "MacroDroid" 安装，或从[官网](https://www.macrodroid.com/)下载 APK。

## 核心概念

| 概念 | 说明 |
|------|------|
| 触发器 | 启动宏的条件（时间、地点、电量、通知、按钮等） |
| 动作 | 触发后执行的操作（发短信、开关 Wi-Fi、启动应用等） |
| 约束 | 限制宏执行的条件（仅工作日、仅充电时等） |

## 使用示例

| 场景 | 配置 |
|------|------|
| 到家自动开 Wi-Fi | 触发器：连接指定基站 → 动作：开启 Wi-Fi |
| 低电量警告 | 触发器：电量 ≤ 15% → 动作：发送通知 |
| 自动静音 | 触发器：日历事件"会议" → 动作：切换静音模式 |
