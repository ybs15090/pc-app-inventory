# 说明
本文件夹收录的是第三方 Android 应用商店，适用于无法访问 Google Play 或需要替代应用分发渠道的场景。

## 清单

- [Aurora Store](Aurora-Store.md) — 开源 Google Play 客户端，支持匿名登录，无需 GMS
- [Gbox](Gbox.md) — 内置 GMS 框架的虚拟容器，无需刷机即可运行谷歌应用（实测好用）
- [APKPure](APKPure.md) — 第三方应用商店，提供 APK/XAPK 下载，无区域限制
- [Uptodown](Uptodown.md) — 国际化应用市场，保留历史版本，无需注册
