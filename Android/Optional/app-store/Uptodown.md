# <img src="../../../.media/png/Uptodown.png" width="60" alt="logo"> Uptodown

Uptodown 是一个国际化的 Android 应用市场，以收录海量应用并提供安全、可追溯的 APK 下载为核心特色。平台对每个上传的 APK 进行病毒扫描和安全校验，并为应用保留完整的历史版本，用户可根据需要自行选择安装某个特定版本。

> Uptodown 同时提供 Android 客户端与网页版，无需注册即可下载。

## 安装

从 [Uptodown 官网](https://uptodown.com/) 或各大应用商店搜索 "Uptodown" 下载客户端。也可直接下载 APK 侧载安装。

## 特点

| 特点 | 说明 |
|------|------|
| 全球应用收录 | 收录来自全球各地的应用和游戏，包括区域限定应用 |
| 版本历史 | 为每个应用保留所有历史版本，支持下载任意旧版 |
| 安全扫描 | 所有 APK 上传后经过反病毒引擎扫描，标注验证状态 |
| 无需注册 | 浏览和下载均不需要账号，开箱即用 |
| 多平台覆盖 | 除 Android 外也收录 Windows、macOS 软件 |

## 使用提示

- **旧版回退**：如果新版应用出现问题，可在 Uptodown 中搜索应用 →「旧版本」→ 选择需要的版本下载。
- **更新通知**：Uptodown 客户端可监控已安装应用的更新，但更新来源为 Uptodown 服务器而非 Google Play，版本推送可能略有延迟。
- **安全性**：下载时注意应用详情页的「已验证」标识和用户评分，优先选择签名校验通过、用户评价良好的版本。
