# Pandoc 
Pandoc 是一个功能非常强大的文档格式转换工具, 支持多种输入和输出格式, 包括 Markdown, HTML, LaTeX, PDF 等等.
例如:
```
Markdown → PDF
Markdown → Word
Markdown → HTML
Word → Markdown
LaTeX → PDF
HTML → EPUB
```

## 安装
在 **Linux** 系统上，可以使用包管理器安装 Pandoc。
```bash
sudo apt-get update
sudo apt-get install pandoc
```
**Windows** 用户可以从 Pandoc 的官方网站下载预编译的二进制文件，并将其添加到系统路径中。


## 使用
官网文档：https://pandoc.org/

Pandoc 的使用非常简单, 例如将 Markdown 文件转换为 PDF:
```bash
pandoc input.md -o output.pdf
```
Tips: 需要安装 LaTeX 发行版（如 TeX Live 或 MiKTeX）才能将 Markdown 转换为 PDF。
推荐安装 [MiKTeX](https://miktex.org/download) 

或者将 Markdown 文件转换为 Word 文档:
```bash 
pandoc input.md -o output.docx
```
