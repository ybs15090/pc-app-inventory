# AI Agent（CLI 工具）

收录面向开发者的终端 AI 编程助手，均以 CLI 方式运行，可读取项目文件、编写代码、执行命令。

## 清单

| 工具 | 开发方 | 简介 |
|------|--------|------|
| [Claude Code](claude-code.md) | Anthropic | 基于 Claude 模型的终端 AI 编程助手，支持文件读写、Shell 执行、Git 操作、MCP 工具集成 |
| [OpenAI Codex CLI](codex.md) | OpenAI | 基于 GPT/o 系列模型的终端 AI 编程助手，支持沙箱模式和多模型切换 |

## 选择建议

- 如果你已有 **Anthropic API 订阅**或偏好 **Claude 系列模型** → 使用 [Claude Code](claude-code.md)
- 如果你已有 **OpenAI API 订阅**或偏好 **GPT/o 系列模型** → 使用 [Codex CLI](codex.md)
- 两者功能高度重叠，也可同时安装，在不同项目/任务中灵活切换
