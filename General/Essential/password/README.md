# 说明
本文件夹收录的是跨平台密码管理与凭据存储工具。

## 清单

- [Bitwarden](bitwarden.md) — 开源跨平台密码管理器，支持自托管
- [Bitwarden Authenticator](bitwarden-authenticator.md) — 独立两步验证（TOTP）应用
- [KeePass](keepass.md) — 本地数据库型开源密码管理器，适合离线保存和便携使用
- [KeePassXC](keepassxc.md) — KeePass 兼容的原生跨平台密码管理器，支持浏览器集成
