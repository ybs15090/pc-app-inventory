# Vim 系列
## <img src="../../.media/svg/gvim.svg" width="40" height="40" alt="gvim Logo"> vim-gtk3 （推荐）
有图形化界面的 vim
安装：
```bash
sudo apt update
sudo apt install vim-gtk3
```
插件: vim-airline
### .vimrc 配置
" ==========================================
"  🎨 界面美化配置 (复制下面的内容)
" ==========================================

" 1. 基础显示优化
```
set number            " 显示行号
set relativenumber    " 显示相对行号 (方便跳转)
set cursorline        " 高亮当前行
syntax on             " 开启语法高亮 (代码变色)
set encoding=utf-8    " 防止中文乱码
```

" 2. 启用鼠标 (在 Gvim/终端中都好用)
```
set mouse=a           " 允许鼠标点击跳转、拖拽窗口
```

" 3. 启用 vim-airline 状态栏美化
" 开启 airline 的标签页栏 (显示打开的文件)
`let g:airline#extensions#tabline#enabled = 1`
" 开启 powerline 字体支持 (如果终端乱码，可注释掉下面这行)
`let g:airline_powerline_fonts = 1`
" 设置状态栏主题为自动或 dark
`let g:airline_theme='dark'`

" 4. 配色方案 (二选一)
" 方案A: 使用 Vim 内置的经典暖色主题 (推荐新手，无需下载)
```
colorscheme desert
set background=dark
```

" 方案B: 如果你喜欢复古护眼风格 (需先安装 gruvbox 插件)
" colorscheme gruvbox
" set background=dark
