- **apt-xapian-index**
是 Linux 系统中通过搜索引擎技术为 APT 软件包构建全文索引的后台工具，主要用于为新立得（Synaptic）等包管理器提供快速、智能的高级搜索与标签筛选功能。

- **deborphan**
是一个用于在 Debian 系统中查找未安装的依赖包的工具，可以帮助你找到那些没有被安装的软件包，以便进行安装或卸载操作。


## AppImageLauncher
### 说明

AppImageLauncher 是一个让 AppImage 绿色版软件能够像标准安装软件一样，自动无缝融入 Linux 桌面菜单和文件关联系统的智能管理工具。
最好**不要**与[Distrobox容器软件](../扩展软件/Distrobox容器软件.md)一起使用（二者不要同时存在于设备上）！！！

### 安装
1. 使用官方维护的 PPA 软件源
    ```bash
    # 添加官方 PPA 软件源
    sudo add-apt-repository ppa:appimagelauncher-team/stable

    # 更新本地包索引
    sudo apt update

    # 安装 AppImageLauncher
    sudo apt install appimagelauncher
    ```
2. 使用官方维护的 PPA 软件源
   **注意：** 如果你的系统版本非常新，或者你使用的是纯净的 Debian，PPA 可能未及时跟进。可以直接去其 GitHub 下载对应的 .deb 安装包，然后使用 `sudo apt install ./appimagelauncher_xxx.deb` 进行本地安装。
