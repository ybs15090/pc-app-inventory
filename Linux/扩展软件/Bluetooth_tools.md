# Blueman <img src="../../.media/svg/blueman.svg" width="35" height="35" alt="Blueman Logo">

Blueman 是一款为 Linux 系统设计的蓝牙管理工具，基于 BlueZ 协议栈（Linux 官方蓝牙协议实现）开发，提供图形化界面（GUI）和命令行（CLI）两种操作方式。它的核心目标是简化蓝牙设备的日常管理，功能覆盖设备配对、文件收发、音频路由、网络共享等场景。

## 安装
在基于 Debian 的系统（如 Ubuntu）上，可以使用以下命令安装 Blueman：

```bash
sudo apt update
sudo apt install blueman -y     #依赖 bluez
```
