# <img src="../../.media/svg/gimp.svg" width="60" height="60" alt="gimp logo"> GIMP

GIMP（GNU Image Manipulation Program）是一款免费开源的图像编辑软件，功能定位为 Photoshop 的开源替代。支持图层、滤镜、色彩调整、批量处理等专业功能，适用于照片修饰、图像合成、图标设计等场景。

## 安装

```bash
# APT（Ubuntu 默认源）
sudo apt update
sudo apt install gimp -y

# Snap（获取更新版本）
sudo snap install gimp --classic
```

## 使用

启动后即可打开图像进行编辑：

```bash
gimp
```

界面布局与 Photoshop 类似，左侧工具箱、右侧图层/通道面板。常用功能包括裁剪、调色、抠图（前景提取）、图层蒙版等。
