# <img src="../../../.media/svg/spark-store.svg" width="60" height="60" alt="Spark Store logo"> 星火应用商店

## 简介

星火应用商店（Spark-Store）是国内领先的 Linux 应用分发平台，由深度社区发起，旨在解决 Linux 桌面用户获取国产应用困难的问题。收录了大量国内常用软件（微信、QQ、WPS、网易云音乐等），提供开箱即用的打包体验。

- [官网](https://www.spark-app.store/)
- [GitHub](https://github.com/spark-store-project/spark-store)

**支持架构：** amd64（x86_64）、arm64（aarch64），实验性支持 loong64。不支持 32 位系统。

## 安装

### Ubuntu / Debian 系

**Ubuntu 22.04+ / Debian 12+** 可直接下载 DEB 包安装：

```bash
# 从官网下载最新 DEB 包
cd ~/Downloads
sudo apt install ./spark-store_*.deb
```

**Ubuntu 20.04** 需先安装依赖包：

```bash
wget https://gitcode.com/spark-store-project/spark-store-dependencies/releases/download/v1.0/deps-ubuntu20.04.deb
sudo apt install ./deps-ubuntu20.04.deb ./spark-store_*.deb
```

如果出现依赖错误，执行：

```bash
sudo apt -f install
```

**添加官方源（推荐，支持自动更新）：**

```bash
sudo bash -c "$(curl -fsSL https://gitcode.com/spark-store-project/spark-store/raw/main/tool/install-repo.sh)"
sudo apt update
sudo apt install spark-store -y
```

### 其它发行版（通过 Distrobox）

Fedora、Arch、openSUSE 等非 Debian 系发行版，可通过 Distrobox 容器运行：

```bash
# 创建 Ubuntu 容器
distrobox create --image ubuntu:24.04 --name box-spark
distrobox enter box-spark

# 在容器内安装
sudo apt install ./spark-store_*.deb
```

容器内如果点击安装无反应，修复 pkexec：

```bash
echo '#!/bin/sh
sudo "$@"' | sudo tee /usr/local/bin/pkexec
sudo chmod +x /usr/local/bin/pkexec
```

## 使用

1. 启动星火应用商店，首次打开会自动刷新软件源
2. 搜索或浏览所需应用 → 点击安装
3. 可在设置中切换国内镜像源（阿里云/腾讯云/华为云）提升下载速度

> **提示：** 星火商店部分应用以 Wine 方式打包运行 Windows 软件（如微信），首次启动可能较慢，需等待 Wine 环境初始化。

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 缺少依赖（如 `libfuse2`） | `sudo apt -f install` |
| 下载速度慢 | 设置 → 切换到国内镜像源 |
| 中文字体显示异常 | `sudo apt install fonts-noto-cjk` |
| 搜索框无法输入中文 | 安装 `fcitx5-frontend-all` 或 `ibus-gtk` |
| 应用安装后不显示图标 | 注销重新登录 |
