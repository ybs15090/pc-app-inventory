# <img src="../../.media/svg/gparted.svg" width="60" height="60" alt="gparted logo"> GParted

GParted 是一款 Linux 下的图形化磁盘分区管理工具，支持创建、删除、调整大小、移动和格式化磁盘分区，兼容 ext2/3/4、NTFS、FAT16/32、XFS 等多种文件系统。

## 安装

```bash
sudo apt update
sudo apt install gparted -y
```

## 使用

启动后以 root 权限运行，选择目标磁盘即可进行分区操作：

```bash
sudo gparted
```

> 分区操作存在数据丢失风险，操作前请务必备份重要数据。

## 相关链接

- [GParted 官网](https://gparted.org/)

# ncdu

ncdu（NCurses Disk Usage）是一款基于终端的磁盘空间分析工具，以类似文件管理器的交互界面逐级展示各目录占用空间，便于快速定位占用空间过大的文件和文件夹。

## 安装

```bash
sudo apt update
sudo apt install ncdu -y
```

## 使用

直接运行即可扫描当前目录，也可指定目标路径：

```bash
ncdu /home
```

常用快捷键：

| 按键 | 功能 |
| --- | --- |
| `↑` / `↓` 或 `k` / `j` | 上下移动选择 |
| `Enter` | 进入所选目录 |
| `<` / `-` | 返回上级目录 |
| `d` | 删除所选文件或目录 |
| `q` | 退出程序 |

> `d` 会直接删除文件，且不可恢复，操作前请确认选中项无误。
