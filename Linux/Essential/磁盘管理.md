# <img src="../../.media/svg/gparted.svg" width="60" height="60" alt="gparted logo"> GParted

GParted 是一款 Linux 下的图形化磁盘分区管理工具，支持创建、删除、调整大小、移动和格式化磁盘分区，兼容 ext2/3/4、NTFS、FAT16/32、XFS 等多种文件系统。

## 安装

```bash
sudo apt update
sudo apt install gparted -y
```

## 使用

启动后以 root 权限运行，选择目标磁盘即可进行分区操作：

```bash
sudo gparted
```

> 分区操作存在数据丢失风险，操作前请务必备份重要数据。
